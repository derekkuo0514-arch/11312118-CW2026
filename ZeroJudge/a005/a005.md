a005.  Eva 的回家作業
# 解題思路
利用len(num) == 1 將第一個固定數字"1"掠過漸少運算需求 ， //是整除可以壁面算術到小數而導致錯誤
# 解題編號
18095991
# 程式碼
```python=
import sys

for line in sys.stdin:
    line = line.strip()
    if not line:
        continue
    nums = list(map(int, line.split()))
    if len(nums) == 1:
        continue
    a, b, c, d = nums
    if b - a == c - b:
        diff = b - a     
        e = d + diff  
    else:
        ratio = b // a   
        e = d * ratio   
    print(f"{a} {b} {c} {d} {e}")
