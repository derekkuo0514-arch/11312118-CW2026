# 题目
a004. 文文的結婚
# 解題思路
在將年份由字串轉換的過程中不可使用map因為map是把每個數拆開變成int型態，這裡要直接使用year = int(輸入年分) 這樣才不會把年份拆成多個數字，要利用sys將輸入數字以外的空格消除避免系統無法辨識

# 解題編號
18095879
# 程式碼
```python=
import sys

for line in sys.stdin:
    line = line.strip()
    if not line:
        continue
    year = int(line)
    
    if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
        print("閏年")
    else:
        print("平年")
